## 💡 1. 핵심 개념
* **함수**: 특정 작업을 수행하는 명령어들의 모음에 이름을 붙인 것 (재사용 가능).
* **지역 변수 (Local Variable)**: 함수 내부에서 생성되어 **함수 안에서만** 사용되는 변수.
* **전역 변수 (Global Variable)**: 함수 외부에서 생성되어 **프로그램 전체**에서 사용되는 변수.
  * 함수 안에서 전역 변수 값을 변경하려면 `global` 키워드 선언이 필요함.

<br>

## 🐍 2. 함수 문법

### 🔹 함수 정의 및 호출
```python
# 함수 정의 (매개변수 x, y)
def add(x, y):
    result = x + y
    return result  # 결과값 반환

# 함수 호출 (인수 10, 20)
print(add(10, 20))  # 출력: 30
