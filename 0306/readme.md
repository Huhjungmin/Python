# 📘 제1장 프로그래밍과 파이썬 소개

## 1. 핵심 개념
* **컴퓨터와 프로그램**: 컴퓨터는 **소프트웨어(프로그램)**만 바꾸면 다양한 일을 하는 유연한 기계이며, 프로그램은 컴퓨터에 내리는 **'작업지시서'**입니다.
* **컴파일러/인터프리터**: 사람이 작성한 코드를 컴퓨터가 이해할 수 있는 **기계어로 번역**해 주는 통역사 역할을 합니다.

## 2. 파이썬(Python) 특징
* **장점**: 문법이 간결하여 **배우기 쉽고 생산성이 높음**.
* **방식**: 코드를 한 줄씩 즉시 실행하는 **인터프리터 언어**.
* **환경**: 데이터 분석 라이브러리가 포함된 **아나콘다(Anaconda)**와 **스파이더(Spyder)** IDE를 주로 사용.

## 3. 기초 문법 및 실행 모드
* **문자열**: 텍스트는 반드시 **따옴표(`" "` 또는 `' '`)**로 감싸야 함.
* **화면 출력**: `print()` 함수를 사용하며, 쉼표(`,`)로 여러 값을 나열 가능.
* **실행 모드**:
  1. **대화형 모드**: 콘솔창에서 한 줄씩 즉시 테스트.
  2. **스크립트 모드**: 파일(`.py`)로 저장하여 전체를 한 번에 실행.
