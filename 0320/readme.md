## 💡 1. 핵심 개념

* **제어 구조**
  1. **순차**: 위에서 아래로 순서대로 실행
  2. **선택 (조건문)**: 조건에 따라 실행할 코드가 바뀜
  3. **반복**: 조건이 만족될 때까지 되풀이
* **들여쓰기 (Indentation)**: 파이썬은 **들여쓰기**로 조건문 안의 코드 블록을 구분함 (칸수가 다르면 오류 발생)

<br>

## 🐍 2. 연산자 요약

### 🔹 관계 연산자 (대소 비교)
* `==` (같다), `!=` (다르다)
* `>`, `<`, `>=`, `<=`

### 🔹 논리 연산자 (조건 결합)
* `and`: 둘 다 참이어야 참
* `or`: 하나만 참이어도 참
* `not`: 참·거짓을 반대로 뒤집음

<br>
