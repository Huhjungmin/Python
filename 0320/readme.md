## 💡 1. 핵심 개념
* **제어 구조**: 순차(차례대로), 선택(조건에 따라), 반복(되풀이).
* **들여쓰기**: 파이썬은 들여쓰기로 코드 블록을 구분 (틀리면 오류).

<br>

## 🐍 2. 연산자
* **관계 연산자**: `==`(같다), `!=`(다르다), `>`, `<`, `>=`, `<=`
* **논리 연산자**: `and`(둘 다 참), `or`(하나만 참), `not`(반대로 뒤집음)
