# 🚀 제2장 변수와 수식

<br>

## 💡 1. 핵심 개념
* **변수**: 데이터를 저장하는 메모리 공간 (값 변경 가능).
* **상수**: 프로그램 실행 중 값이 변하지 않는 수.
* **주석 (`#`)**: 코드를 설명하는 글 (프로그램 실행에 영향 없음).

<br>

## 🐍 2. 산술 연산자
* `+`, `-`, `*`, `/` : 사칙연산
* `//` : **몫** (`10 // 3` $\rightarrow$ `3`)
* `%` : **나머지** (`10 % 3` $\rightarrow$ `1`)
* `**` : **거듭제곱** (`10 ** 3` $\rightarrow$ `1000`)

<br>
