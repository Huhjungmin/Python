## 💡 1. 변수 (Variable)와 상수 (Constant)

* **변수 (Variable)**
  * 컴퓨터의 **메모리 공간에 이름을 붙인 것**으로, 데이터를 저장하는 상자와 같습니다.
  * 변수에 저장된 값은 언제든지 변경할 수 있습니다.
* **상수 (Constant)**
  * 프로그램이 실행되는 동안 **값이 변하지 않는 수**를 의미합니다. (예: `3.141592`)
* **주석 (Comment)**
  * 사람을 위한 설명 글로, 파이썬 인터프리터는 무시합니다.
  * 한 줄 주석은 샵(`#`) 기호로 시작하며, 프로그램의 가독성을 높이는 데 필수적입니다.

<br>

## 🐍 2. 연산자와 수식

파이썬은 다양한 산술 및 할당 연산자를 제공하며, 수학 연산과 유사한 우선순위를 따릅니다.

### 🔹 산술 연산자 요약
| 연산자 | 기능 | 예시 | 결과 |
| :---: | :--- | :--- | :--- |
| `+` / `-` | 더하기 / 빼기 | `10 + 3` | `13` |
| `*` / `/` | 곱하기 / 나누기 | `10 / 3` | `3.3333...` |
| `//` | **몫** 계산 | `10 // 3` | `3` |
| `%` | **나머지** 계산 | `10 % 3` | `1` |
| `**` | **거듭제곱 (제곱)** | `10 ** 3` | `1000` |

### 🔹 할당(대입) 연산자
* `=` 기호는 오른쪽 수식의 계산 결과를 왼쪽 변수에 저장(대입)합니다.
  * 예: `x = 10` (변수 x에 10을 저장)
* **복합 대입 연산자**: `x += 1`은 `x = x + 1`과 완전히 동일하며 코드를 간결하게 만듭니다. (`-=`, `*=`, `/=` 등도 가능)
