## 📂 제8장 객체와 클래스
* **객체 지향 프로그래밍 (OOP)**: 데이터(속성)와 함수(동작)를 하나의 덩어리로 묶은 **객체(Object)**를 조립하여 프로그램을 만드는 방식.
* **클래스 (Class)와 객체**: 클래스는 객체를 만들기 위한 **설계도(틀)**이며, 이를 통해 실제로 메모리에 생성된 실체를 **인스턴스(객체)**라고 함.
* **생성자 (`__init__`)**: 객체가 생성될 때 속성(변수)을 초기화하기 위해 자동으로 호출되는 특수 메서드.
* **`self` 키워드**: 메서드 내부에서 현재 생성된 객체 자신을 가리키는 첫 번째 매개변수.
