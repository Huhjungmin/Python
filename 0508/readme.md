## 💡 1. 핵심 개념
* **자료구조 (Data Structure)**: 데이터를 효율적으로 저장하고 관리하는 구조.
* **시퀀스 (Sequence)**: 요소들이 순서를 가지고 나열된 구조 (튜플, 문자열 등).
